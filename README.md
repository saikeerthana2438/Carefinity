# 🩺 Carefinity – AI Powered Smart Healthcare Companion

> **"One Application. Complete Healthcare Management."**

Carefinity is an AI-powered healthcare companion developed using **Flutter**, **Supabase**, and **Google Gemini AI**. It brings together healthcare services, AI-powered medical assistance, diagnostics, medical report management, women's health tracking, family healthcare, and wellness monitoring into one intelligent mobile application.

Our mission is to simplify healthcare by providing users with a secure, intelligent, and easy-to-use platform that helps them manage every aspect of their health from a single application.

---

# 🌟 Problem Statement

Healthcare today is fragmented across multiple applications and physical records.

Users often need separate apps to:

- Book doctor appointments
- book  diagnostic tests
- Store medical reports
- Track health activities
- Manage menstrual cycles
- Receive medication reminders
- Receive appointments diagnostic tests reminders
- Maintain family medical records

This makes healthcare management complicated and inefficient.

---

# 💡 Our Solution

Carefinity integrates all major healthcare services into one platform powered by Artificial Intelligence.

With Carefinity, users can:

- 👨‍⚕️ Consult doctors
- 🧪 Book diagnostic tests
- 📄 Store medical reports securely
- 🤖 Analyze reports using AI
- 💬 Chat with an AI Health Assistant
- 🌸 Track menstrual health
- 🚶 Participate in wellness challenges
- 👨‍👩‍👧 Manage family health profiles
- 🎤 Navigate using voice assistance

Everything healthcare, all in one place.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/your-username/carefinity.git
```

## Navigate to Project

```bash
cd carefinity
```

## Install Dependencies

```bash
flutter pub get
```

## Run the Application

```bash
flutter run
```
